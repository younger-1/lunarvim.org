# Updating LunarVim

Update your plugins, LunarVim and neovim:

# Plugins

In lvim:

:PackerUpdate
1

# LunarVim

cd ~/.local/share/lunarvim/lvim && git pull
1

Then, in lvim:

:PackerSync
1

# Neovim

Use your package manager or compile from source.
